Getting started with Steam

Let's get set up with Steam and get our API key.

First, you will need to sign up for a free steam account in order to receive your API key.

Visit the page here and click on the "Join Steam" button located on the page.

After you have created an account. You will be taken to the page where you will receive your key. In the section where it asks for the domain you will be using, it is ok to just enter "localhost" for now.

After you agree to the Steam Web API terms of use and click "Register", you will be shown your API key. Be sure to keep this key safe and private! You don't want anyone else getting a hold of this.

Right now, you may be asking youself, "What is an API key and why do I need one to get started?" Well, you can think of an API key as a hall pass. Just as the hall pass allows you to go through the halls to get to a destination without problems, an API key let's an application know that it is ok for you to make requests to its domain, and it is ok for that application to send back data to you.
