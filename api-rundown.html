What is an API?

What exactly is an API anyway?

What does API stand for?

API stands for Application Programming Interface. It is what we can use in a program when we want to access different sorts of data without the user knowing what is going on behind the scenes.

We use an API to make "calls" to a resource and that resource "responds" with the data that we requested. Let's break this down a little bit further.

Application

In today's world, you more than likely interact with applications everyday when using your smartphone. Software such as facebook, twitter, and games on your phone are just some of the examples of applications that we interact with on a daily basis.

Programming

Programming is what software engineers do everyday that allow us to use all of those great applications on our phone. More than likely, if an engineer needs to display data in an application, they are probably taking advantage of some sort of API.

Interface

An interface is a common connection shared by two applications that allows programmers to communicate with those applications.
